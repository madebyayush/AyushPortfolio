## ⚡ Running the Portfolio Locally

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/madebyayush/AyushPortfolio.git
   cd AyushPortfolio
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Start Development Server:**
   ```bash
   npm run dev
   ```

4. **Build for Production:**
   ```bash
   npm run build
   ```
